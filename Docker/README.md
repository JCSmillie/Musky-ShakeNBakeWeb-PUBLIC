# Musky Docker Notes

The preferred Docker path is now the repo-root compose setup plus the new quick-start guide:

- `Docs/README.DockerQuickStart.md`

That flow gives you:

- Nora Docker first
- Musky Docker second
- MariaDB-first setup
- browser-based first-time local admin bootstrap
- no new SQLite dependency for clean installs

## What stays in this folder

This directory still contains earlier Devilbox/package material for reference:

- historical README notes
- the old packaged zip
- early bundle assets used during that prototype stage

That material is archival now. It is not the recommended deployment path for a new Musky setup.
