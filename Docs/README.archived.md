# archived/

This folder contains deprecated, outdated, or superseded modules from the Musky DeviceManager system.

These modules are retained for historical or reference purposes but are not expected to run in production.

## 🔒 Rules for Archived Modules

- Do not re-enable without code review.
- Do not include in autoloaders or menus.
- If reactivating, move to `/Modules/` and rename appropriately.

## 📦 Why Keep Them?

- They may contain useful parsing patterns or legacy compatibility tricks.
- They show how previous workflows were implemented.
- Helpful for training new devs or debugging regressions.
