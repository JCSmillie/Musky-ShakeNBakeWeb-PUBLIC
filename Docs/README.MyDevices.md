# 📱 MyDevices – User Guide (Musky)

MyDevices lets you quickly see the iPads assigned to you and take safe, guided actions on them.  
This page is part of the Musky Device Manager suite.

---

## ⭐ What MyDevices Does

- Shows **all iPads assigned to you**
- Displays:
  - Model & generation
  - OS version
  - District Asset Tag
  - Last check-in time
  - Device health status (battery, storage, update needs, etc.)
  - Decoded device tags (e.g., LOSTMODE, NEEDSUPDATE, REPAIR)
- Lets you take actions on a selected device:
  - **Soft Reboot**
  - **Open Device Report**
  - **Open Ticket** for issues
- Admin users can:
  - Search any user
  - Power Reboot
  - Power Wash & Wax (MDM Wipe)

---

## 🎯 How to Use MyDevices (Students & Staff)

### 1. **Find your devices**
When you open MyDevices, your assigned iPads appear automatically.

### 2. **Select ONE device**
Click on a device block.

- The selected device gets highlighted.
- All other devices fade, so you know which one is active.
- The action buttons on the right will unlock.

👉 **Important**: You *must* select a device to enable the actions.

---

## 📌 Actions Available

### 🟦 Report a Problem
Opens a ticket form pre-linked to the selected device.

Use this when:
- Your device is not working correctly  
- A screen is cracked  
- Wi-Fi stops working  
- You need technical help  

### 🔁 Soft Reboot Device
Safely restarts your iPad through Mosyle MDM.  
Most problems disappear after a reboot.

### 📊 Device Report
Opens a detailed diagnostic report showing:
- Model & generation  
- OS version  
- Battery level  
- Disk usage  
- Update status  
- Management status  
- Recent check-in information  

---

## 🔄 Refresh
Click **Look Up Again** to reload your data.

This is useful when:
- You just restarted a device  
- You reconnected it to Wi-Fi  
- You want to see last check-in updates  

---

## 🧰 Admin-Only Features

Admins with **DEVICE_MANAGER** or **ALL_TOOLS** see extra tools:

### 🔍 Search Another User
Enter:
- a username (ex: `jsmith`)  
- or full email (ex: `jsmith@example.org`)  

MyDevices will show their assigned devices.

### ⚡ Power Reboot  
Forcibly reboots the selected iPad.

### ☣ Power Wash & Wax  
Performs a full wipe (reset) for the selected iPad.

> ⚠ Admin tools are only enabled when a device is selected.

---

## 📝 Tips for Best Use

- Click a device **first**, then select your action.  
- If something looks wrong, reload using **Look Up Again**.  
- If no devices appear, your account may not be linked yet.  
- Students do **not** get admin tools.

---

## 🖼️ Icons Used

- 🟦 Blue Badge = Normal device health  
- 🟧 Orange Badge = Some issue detected  
- 🚨 “CONTACT MR. SMILLIE ASAP!!” = Device is tagged STOLEN
